Preparations

Place the ramanlibs folder in a directory that Python searches for imports. This will likely be a "site-packages" folder in your user profile or the directory where your own scripts are stored. Whereever using this module, place proper import statements in the preamble of your code:

import ramanlibs.sif as sif # and/or
import ramanlibs.avantes as avantes

For all examples, we will assume this import-as alias to be used.

General purpose helper functions

Usually no need to take of them. They are internally used to parse directories for suitable files and identify measurement details in file names. If you want to retrieve the list of files instead of their contents, feel free to import these parts with
import ramanlibs.general
Then you can e.g. get the list of files and the index i in the respective path where the file name starts with
files, i = ramanlibs.general.getFileList(path, pattern, **options)

The sif-package

### Remember import ramanlibs.sif as sif ###

The sif sub-package (hopefully) works separate from the avantes function to avoid cluttering the memory with unnecessary avantes calibration data and competence. It provides functions to load single files and whole directories of .sif files into memory, basic operations on sif data, and the underlying Sif class holding the data.

Functions

sif.load(path, pattern="*.sif", **options)
This function loads single files and returns a Sif object, or all files in ⟨path⟩ matching ⟨pattern⟩ and returns a list of Sif objects.
